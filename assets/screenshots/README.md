# 截图清单

把截图放到这个目录，建议使用下面的文件名。

## 必备截图

1. `01-home.png`
   - 小程序首页或工作台。
   - 展示入口：JD 分析、问一问 AI、复盘、历史。

2. `02-jd-result.png`
   - JD 分析结果页。
   - 尽量截到匹配分、风险点和高风险面试雷达。

3. `03-coach-result.png`
   - 问一问 AI 回答页。
   - 尽量截到完整答法、可引用证据、可能追问。

4. `04-review-memory.png`
   - 复盘页或 Job Memory 生成结果。
   - 展示系统如何把面试复盘结构化。

5. `05-history-detail.png`
   - 历史页展开详情。
   - 展示面试问题、卡点、下次策略、可复用证据。

## 截图建议

- 使用微信开发者工具模拟器截图即可。
- 不要截真实 API Key、邀请码、服务器后台、`.env`。
- 如果页面里有真实公司名或个人信息，先替换成演示数据。
- 尽量保持同一设备尺寸，视觉会更统一。
